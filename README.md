# タスク管理アプリ

シンプルで使いやすいタスク管理アプリケーションです。

## 機能

- ✅ タスクの追加・削除
- ✅ タスクの完了/未完了の切り替え
- ✅ ローカルストレージへの自動保存
- ✅ リアルタイム統計表示（全タスク数、完了済み、残り）
- ✅ レスポンシブデザイン

## ビルド方法

このアプリはHTML5で実装されており、ビルドは不要です。

### 実行方法

1. `index.html` をブラウザで直接開く、または
2. ローカルサーバーで実行：

```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx http-server

# その他のサーバー
# お好みのローカルサーバーで実行してください
```

3. ブラウザで `http://localhost:8000` にアクセス

## 使い方

1. テキスト入力欄に新しいタスクを入力
2. 「追加」ボタンをクリックするか、Enterキーを押す
3. タスク左側のチェックボックスをクリックして完了/未完了を切り替え
4. 「削除」ボタンでタスクを削除

すべてのタスクはブラウザのローカルストレージに自動保存されます。
